class changegen.changewriter.OSMChangeWriter(filename=None, compress=False)¶

Bases: object

Write OSMChange format (https://wiki.openstreetmap.org/wiki/OsmChange) to file_like object. close() MUST be called to ensure compliance with XML schema.

Provides support for modify and add tags currently, with support for Node, Way, and Tag OSM elements (defined above.)

add_create(elementlist)¶

Creates <create> element containing all elements in elementlist.

NOTE: does not ensure that <node> elements are added for every <node> in any Ways in the elementlist.

add_delete(elementlist)¶

Creates a <delete> element containing all elements in elementlist

add_modify(elementlist)¶

Creates <modify> element containing all elements in elementlist.

close()¶

Add the <osmChange> closing tag and close the file.

changegen.changewriter.write_osm_object(osm, writer)¶

Writes an OSM object (Node, Way) as an XML object. Uses the type of the namedtuple representing each OSM object to determine the type of XML element to write.

Creates child elements for each Tag or nd (node ref)
